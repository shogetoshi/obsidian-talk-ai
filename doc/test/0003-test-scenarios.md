**目的**: 設定画面に新しい設定項目が表示されることを確認

**前提条件**:

**手順**:
1. Obsidianの設定を開く
2. プラグイン設定から「Talk AI」を選択

**期待結果**:

### TC6: API呼び出しでの設定反映 (システムプロンプトあり)
**目的**: システムプロンプトがAPI呼び出しに正しく含まれることを確認

**前提条件**:
- 有効なAPIキーが設定されている
- AIモデル: GPT-5.2
- システムプロンプト: 「あなたは親切で丁寧な言葉遣いで回答するアシスタントです。」

**手順**:
1. 新しいノートを作成
2. 「JavaScriptとは何ですか?」と入力
3. リボンアイコンをクリックしてAIに質問

**期待結果**:
- ✅ APIリクエストのmessages配列の1番目に以下が含まれる:
  ```json
  {
    "role": "system",
    "content": "あなたは親切で丁寧な言葉遣いで回答するアシスタントです。"
  }
  ```
- ✅ APIリクエストのmessages配列の2番目に質問が含まれる:
  ```json
  {
    "role": "user",
    "content": "JavaScriptとは何ですか?"
  }
  ```
- ✅ APIリクエストのmodelが "gpt5.2" になっている
- ✅ 回答が丁寧な言葉遣いになっている

**検証方法**:
- ブラウザの開発者ツールでネットワークリクエストを確認
