# 実装計画: 0001-任意質問

## 概要
現在、固定の質問「10000までの素数を計算するプログラム書いて」を使用しているプラグインを、現在開いているページから質問を読み込み、指定フォーマットで出力できるように変更します。

## 要件分析

### 現在の実装
- リボンアイコンまたはコマンドから `askAI()` メソッドを実行
- 固定の質問を使用 (line 51)
- 出力フォーマット: `**質問:** {question}\n\n**回答:**\n{AI回答}`
- OpenAI APIをストリーミングで呼び出して回答を取得

### 要求される変更
1. **入力**: 現在開いているページから質問を読み込む
2. **出力フォーマット**: 以下の形式に変更
```
# Q
{質問内容}

---

# A
{AI回答}
```

## 変更が必要なファイル

### 1. main.ts

## 具体的な変更内容

### 1. `askAI()` メソッドの変更 (main.ts: line 35-73)

#### 変更前の処理フロー
1. アクティブなエディタを取得
2. APIキーを確認
3. 固定の質問を使用
4. 質問と回答の見出しをエディタに追加
5. OpenAI APIをストリーミングで呼び出し

#### 変更後の処理フロー
1. アクティブなエディタを取得
2. APIキーを確認
3. **[新規] 現在のエディタから全テキストを読み込み、それを質問として使用**
4. **[変更] 新しいフォーマットで質問をエディタに追加**
5. OpenAI APIをストリーミングで呼び出し

#### 詳細な変更内容

**A. 質問の読み込み (line 50-51を変更)**
```typescript
// 変更前:
const question = '10000までの素数を計算するプログラム書いて';

// 変更後:
const question = editor.getValue(); // 現在のエディタの全テキストを質問として取得
```

**B. 出力フォーマットの変更 (line 53-59を変更)**
```typescript
// 変更前:
const currentCursor = editor.getCursor();
editor.replaceRange(`\n\n**質問:** ${question}\n\n**回答:**\n`, currentCursor);
const answerStartLine = currentCursor.line + 4;
editor.setCursor({ line: answerStartLine, ch: 0 });

// 変更後:
// エディタの内容を全てクリアし、新しいフォーマットで出力
const formattedQuestion = `# Q\n${question}\n\n---\n\n# A\n`;
editor.setValue(formattedQuestion);

// 回答の開始位置を計算 (最終行)
const lastLine = editor.lastLine();
const answerStartLine = lastLine;
editor.setCursor({ line: answerStartLine, ch: editor.getLine(answerStartLine).length });
```

**C. OpenAI APIリクエストの変更 (line 82-91)**
```typescript
// line 87の質問内容を変数に変更
// 変更前:
content: '10000までの素数を計算するプログラム書いて'

// 変更後:
content: question // askAI()から渡される質問を使用
```

## 実装手順

### ステップ1: `streamOpenAIResponse()` メソッドの引数を拡張
1. メソッドシグネチャに `question: string` パラメータを追加
2. line 87のハードコードされた質問を `question` パラメータに置き換え

### ステップ2: `askAI()` メソッドの質問読み込みロジックを変更
1. line 51の固定質問を `editor.getValue()` で置き換え
2. 読み込んだ質問が空の場合のバリデーションを追加
   - 空の場合は Notice を表示して処理を中断

### ステップ3: 出力フォーマットの変更
1. line 53-59を新しいフォーマットに置き換え
   - `editor.setValue()` で全体を置き換え
   - `# Q`, `---`, `# A` の形式で出力

### ステップ4: `streamOpenAIResponse()` 呼び出しの更新
1. line 65の呼び出しに `question` パラメータを追加

### ステップ5: エラーハンドリングの追加
1. 質問が空文字列の場合のバリデーション
2. 質問が長すぎる場合の警告 (オプション)

## 想定されるエッジケース

### 1. 空のエディタ
- **問題**: エディタに何も書かれていない場合
- **対策**: 質問が空の場合は Notice で警告を表示し、処理を中断

### 2. 非常に長い質問
- **問題**: APIの制限を超える可能性
- **対策**: 現時点では OpenAI API のエラーハンドリングに任せる(既存のtry-catchで対応済み)

### 3. マルチバイト文字
- **問題**: 日本語などの質問
- **対策**: 既存実装で対応済み(UTF-8エンコーディング)
