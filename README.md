# Talk AI - Obsidian Plugin

OpenAI APIを使用してAIとの会話を実現するObsidianプラグインです。

## 機能

- **任意の質問**: エディタ上で自由にAIに質問できます
- **会話履歴**: `# Q` / `# A` 形式で会話コンテキストを維持し、連続した対話が可能です
- **ストリーミング応答**: AIの回答がリアルタイムにエディタへ書き込まれます
- **モデル選択**: 設定画面からAIモデルを切り替えられます（gpt-5.4 / gpt-5.2 / gpt-5.2-pro）
- **システムプロンプト**: AIの振る舞いを制御するシステムプロンプトを設定できます
- **外部モデル設定**: `modelOptions.json` でモデル一覧をコード変更なしに追加・変更できます

## セットアップ

### 1. 依存関係のインストール

```bash
npm install
```

### 2. ビルド

開発モード（自動リビルド）:
```bash
npm run dev
```

プロダクションビルド:
```bash
npm run build
```

### 3. Obsidianへのインストール

1. ビルド後、`main.js`と`manifest.json`を Obsidianのプラグインフォルダにコピーします
   - 場所: `<Vaultフォルダ>/.obsidian/plugins/obsidian-talk-ai/`
2. Obsidianを再起動するか、設定からプラグインをリロードします
3. 設定 → コミュニティプラグイン → Talk AIを有効化します

### 4. APIキーの設定

1. https://platform.openai.com/api-keys でOpenAI APIキーを取得します
2. Obsidianの設定 → Talk AI → OpenAI APIキー に入力します

## 使用方法

### AIへの質問

**方法1: リボンアイコンから**
左サイドバーのメッセージアイコンをクリック

**方法2: コマンドパレットから**
1. `Ctrl/Cmd + P` でコマンドパレットを開く
2. "AIに質問する" を検索して実行

### 会話形式

エディタ上で以下の形式で会話が管理されます:

```markdown
# Q
ここに質問を入力

---

# A
AIの回答がストリーミングで書き込まれます

---

# Q
次の質問を入力（会話履歴が維持されます）
```

質問を `# Q` の下に記入し、コマンドを実行すると、`# A` の下にAIの回答がストリーミングで書き込まれます。回答完了後、次の `# Q` が自動で追加されるため、続けて会話できます。

## 設定項目

| 項目 | 説明 | デフォルト値 |
|------|------|-------------|
| OpenAI APIキー | OpenAI APIの認証キー | （空） |
| AIモデル | 使用するモデル | gpt-5.2 |
| システムプロンプト | AIの振る舞いを制御するプロンプト | （空） |

### モデルの追加

プラグインフォルダ内の `modelOptions.json` を編集することで、コードを変更せずにモデルの選択肢を追加・変更できます。

```json
{
  "gpt-5.4": "gpt-5.4",
  "gpt-5.2": "gpt-5.2",
  "gpt-5.2-pro": "gpt-5.2-pro"
}
```

## 開発

```bash
npm run dev
```

ファイルを変更すると自動的に再ビルドされます。Obsidianでプラグインをリロードして変更を反映してください。
